## Description

This is a frontend prototype for the Grow My Tree project, a platform dedicated to tree planting initiatives focused on species like Sandalwood, Red Sandalwood, and Miyazaki Mango. The prototype demonstrates the user interface and user experience for browsing products, managing a shopping cart, viewing product details, and contacting the organization. It serves as a proof-of-concept for the full application, emphasizing environmental conservation and sustainable agriculture.

**Note:** This is a prototype version of the frontend and is not intended for production use.

## Features

- **Homepage**: Overview of the tree planting initiative with key information and calls to action.
- **Product Catalog**: Browse available tree planting products and investment opportunities.
- **Product Details**: Detailed view of individual products including descriptions and specifications.
- **Shopping Cart**: Add products to cart and manage selections.
- **Contact Page**: Form for users to get in touch with the organization.
- **Responsive Design**: Optimized for desktop and mobile devices.
- **Interactive Elements**: Carousel sliders, animations, and user-friendly navigation.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Framework**: Bootstrap 4/5 for responsive layout
- **Libraries**:
  - jQuery for DOM manipulation
  - Owl Carousel for image sliders
  - PrettyPhoto for lightbox galleries
  - Animate.css for animations
  - Font Awesome for icons
- **Fonts**: Google Fonts (Open Sans, Merriweather, Droid Serif)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/grow-my-tree-prototype.git
   ```

2. Navigate to the project directory:
   ```bash
   cd grow-my-tree-prototype
   ```

3. Open `index.html` in your preferred web browser.

No additional dependencies or build steps are required as this is a static frontend prototype.

## Usage

- Open `index.html` in a web browser to start exploring the prototype.
- Navigate through the different pages using the menu:
  - Home: `index.html`
  - Products: `product.html`
  - Product Details: `product-details.html`
  - Cart: `cart.html`
  - Contact: `contact.html`
- Interact with the UI elements to simulate user actions like adding items to cart or viewing product details.
